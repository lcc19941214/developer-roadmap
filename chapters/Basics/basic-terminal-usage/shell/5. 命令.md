# 5. 命令
## echo

1. 显示字符串
```bash
echo "hello world"
echo hello world
```

2. 显示转义字符
```bash
echo "This is \"Cool Bird\""
```

3. 显示变量
```bash
name="Conan"
echo "hello $name"
```

4. 显示换行
使用-e参数
```bash
echo -e 'hello world\n'
echo 'goodbye'
```
5. 显示命令结果
使用反引号\` \`
```bash
echo `ls`
```

6. 结果重定向
```bash
echo ok > text.txt
```
